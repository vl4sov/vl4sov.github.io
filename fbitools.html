FBI Tools
class="subtext-project">многофункциональный скрипт работающий на базе библиотеки 'MoonLoader', написанный на платформе Lua, целью которого является облегчение работы для сотрудников правоохранительных органов. Включает в себя упрощённую систему выдачи розыска за определённую причину, автоматическую Role Play отыгровку при использовании некоторых команд и/или нажатии определённых кнопок, упрощённый диалог с игроками и красивый интерфейс. А так же многие различные функци которые упрощают работу в государственных организациях.

Список доступных команд:

/ft - Открыть основное меню скрипта

/shp - Открыть шпаргалку

/fnr - Созвать сотрудников организации на работу

/fmask - Быстро натянуть маску ( Оптимизировано под Evolve RP )

/fkv [Сектор] - Установить метку на определённый квадрат

/ticket - Открыть меню выдачи штрафов

/dlog - Открыть лог сообщений рации департамента

/rlog - Открыть лог сообщений фракционной рации

/sulog - Открыть лог объявлений в розыск

/smslog - Открыть лог личных сообщений

/kmdc - Пробить саспекта по базе данных

/su [ID] - Объявить в розыск по выбранной в списке статье

/megaf [ID] или /st [ID/Опционально] - Запросить остановку транспортного средства

/ak - Открыть окно с отображением Административного Кодекса

/yk - Открыть окно с отображением Уголовного Кодекса

/fp - Открыть окно с отображением Федерального Постановления

/warn [ID] - Выдать предупреждение по волне департамента за неправильныую выдачу розыска

/dmb - Открыть окно с отображением /members

/mb - Сокращение команды /members

/blg [ID] [Фракция] [Причина] - Высказать благодарность по волне департамента

/cc - Очистить чат

/df - Открыть меню разминирования бомбы

... - И многие другие команды для облегчения игры

Более подробную информацию можно посмотреть в группе скрипта
